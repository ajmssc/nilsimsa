Nilsimsa for Java
=================
This library provides functions to compute the Nilsimsa hash for a
given string.

Written by Albert Weichselbraun <albert.weichselbraun@htwchur.ch>
                                <weichselbraun@weblyzard.com>
 
The code is a translation of the Python Nilsimsa implementation 
by Michael Itz to the Java language. 
  http://code.google.com/p/py-nilsimsa.
 
Original C nilsimsa-0.2.4 implementation by cmeclax:
 http://ixazon.dynip.com/~cmeclax/nilsimsa.html

Requirements
============
* Java 7 or higher
* Apache Commons libraries from http://commons.apache.org/
